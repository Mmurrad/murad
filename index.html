Project

Current and Past Research Projects

1. P2P Communication in Fog Computing

This project introduces a new integrated peer-to-peer (P2P) architecture within the fog layer. The proposed model is an advanced iteration of the existing 2-layer LDEPTH architecture, specifically tailored for P2P networks. In this sophisticated framework, networks are meticulously organized in a hierarchical manner. Each network within this hierarchy is characterized by a remarkably concise diameter, limited to a single overlay hop. GitHub

2. Job Scheduling Technique in Cloud Computing

In this project, a novel priority-based job scheduling method, designated as SG-PBFS, has been introduced and meticulously developed. Several experiments have been conducted to validate and refine this technique. The SG-PBFS method is innovatively designed, considering several critical factors, including CPU time, the timing of job arrivals, and the length of each job. This approach aims to optimize scheduling efficiency by prioritizing tasks based on these parameters, leading to more effective management of computational resources and enhanced overall system performance. GitHub

3. Department Management System Through an Android App

This project involved developing a sophisticated Android application crafted meticulously using Java. The application is designed to serve as a comprehensive educational management tool, encompassing many functionalities. It allows for the efficient addition and management of various academic elements such as notices, detailed course information, and student profiles. Additionally, the application streamlines the process of taking attendance, thereby enhancing the efficiency of classroom management. It also includes features for recording and analyzing semester results, providing a robust platform for academic assessment. Furthermore, the application facilitates a unique feature for integrating parents' comments, offering a channel for parental engagement in the educational process. GitHub

4. Impact learning Implementation

Impact Learning is a robust machine learning algorithm, which is being used widely to achieve more advanced results on many machine-learning related challenges. Impact learning is a supervised machine learning algorithm for resolving classification and linear or non-linear regression knowledge from examples. To install in your machine learning environment just command pip installs impact learning. GitHub

5. An Intelligent Android to Predict Asthma

Contributed to and engaged in research for the creation of a socially intelligent application, aimed at developing predictive and treatment planning systems for asthma. The work involved researching, designing, and implementing a machine learning strategy focused on minimizing errors during the execution of tasks. This approach was integral to enhancing the application's accuracy and efficiency in managing asthma-related health scenarios.. GitHub

6. Java-Powered Customer Order Management and Analytics System

This application, crafted using Java, is designed to facilitate the process of order placement by customers. It enables users to select and order items with ease. Additionally, the app provides a feature for customers to leave reviews for their purchased items, offering valuable feedback and insights. Beyond these functionalities, the app has a comprehensive system to track and summarize all transactions over a month. This feature provides a clear overview of monthly sales and customer engagement, making it an essential tool for business analytics and strategy development. The application's user-friendly interface and robust backend make it a reliable and efficient solution for managing customer orders and reviews and analyzing monthly business performance. GitHub
